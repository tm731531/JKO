# JKO
# 一個CLI指令集工具

# 開發環境 
- netcore 
- Azure DB

# 專案結構
| 資料夾 | 功能 |備註 | 
| -------- | -------- | -------- |
| JKO     | 主程式     |      |
| JKO.Service | 工廠跟服務主體 |      |
| JKO.Dao     | 資料庫操作   |  |
| JKO.Model     | 物件存放區  |  |
| JKO.DB     | 資料庫相對應SQL|DB pro |

# 使用範例

https://github.com/tm731531/JKO/blob/master/JKO/run.sh



| 指令 | 功用 |
| -------- | -------- |
| REGISTER         |註冊 User|
| CREATE_LISTING   |建立 list|
| GET_LISTING      |取得 list|
| GET_CATEGORY     |取得 分類|
| GET_TOP_CATEGORY |取得 最高排名分類|
| DELETE_LISTING   |刪除 list |
